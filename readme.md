Представляем вашему вниманию гибкий, удобный и самое главное БЕСПЛАТНЫЙ плагин на данжи для вашего Minecraft проекта - DRdung.

Плагин находится в бете, но уже сейчас позволяет настраивать не только ванильные предметы с чарами или без, шансы и количество, но так же даёт возможность загружать свои структуры и модифицированные предметы.

При добавление своих предметов плагин запоминает такие вещи как:
• NBT
• Attribute

При спавне данжа выводится сообщение, которое можно редактировать как вам хочется:
• Смена текста спавна данжа
• Поддержка HEX (Система цветов 1.16+)

Плагин УЖЕ предоставляет 3 данжа, которые готовы к использованию. Плагин автоматически подстраивает данжи под ландшафт и сводит все лаги при их спавне к НУЛЮ.
При необходимости можно настроить генерацию данжей:
• Добавить свою структуру. Так же есть возможность устанавливать несколько схематик
• Минимальный радиус спавна между данжами
• Максимальный радиус спавна от центра мира

В плагин так же включена защита от фейковых ключей, которая не позволяет игрокам создавать свои ключи и использовать их для открытия данжей.

Для управления данжами вне игры использовать конфиг плагина.
Для управления данжами внутри игры есть список необходимых команд для этого:
• /drdung - Открывает меню управлением данжами
• /drdung help - Выводит список допустимых команд
• /drdung reload - Перезапускает плагин
• /drdung spawn - Открывает меню спавна данжей с настройками спавна
• /drdung loot - Открывает меню с настройкой лута
• /drdung key [ИМЯ ИГРОКА] [КОЛИЧЕСТВО] - Выдача ключа (ключей) игроку

Зависимыми части плагина:
• WorldEdit
• WorldGuard
